# Mon Portfolio

Portfolio personnel présentant mes projets et compétences.

## Description

Ce portfolio a été créé pour mettre en valeur mes projets, compétences techniques et mon parcours professionnel.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript

## Installation

```bash
# Cloner le repository
git clone https://github.com/votre-username/mon-portfolio.git

# Accéder au dossier
cd mon-portfolio
```

## Utilisation

Ouvrez simplement le fichier `index.html` dans votre navigateur pour visualiser le portfolio.
